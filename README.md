# project-euler #

[![Build Status][travis-image]][travis-url]

My own tackling of [Project Euler][project-euler] problems in ES6

## Install ##

 * `git clone`
 * `npm install`

## Running tests ##

```bash
npm run test
```

## Validating ##

`npm run validate`
